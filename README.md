# TodoList - SwiftUI

A simple Todo List iOS application built using SwiftUI.

This project demonstrates basic SwiftUI concepts including creating tasks, displaying a checklist, marking tasks as completed, deleting tasks, and managing UI state.

## Features

- View task list
- Add new tasks
- Mark tasks as completed using a checkbox
- Show completed tasks with checkmark and strikethrough
- Prevent duplicate tasks
- Delete tasks using swipe action

## Technologies Used

- Swift
- SwiftUI
- Xcode
- iOS
